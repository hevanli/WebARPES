Hello, Welcome to WebARPES

WebARPES is a work in progress to calculate photocurrents for transition metal-ligand planar clusters in a tight-binding framework, as shown in the figure below. ARPES matrix elements are calculated using hydrogenic orbitals.

I have assumed a work function for Cu=4.46 eV

More details about this web application can be found here.

How To Use:

  1. Go to set parameters.
  2. Click calculate.
  3. Click plot bands to plot ARPES bands through specified cuts.
  4. click plot DOS to plot DOS.
  5. Click plot contour to plot spectral function at fixed energy.
  6. Click ARPES datafile to download ARPES datafile.

Things to be added:
