WebARPES is a work in progress to calculate photocurrents for transition metal-ligand planar clusters in a tight-binding framework, as shown in the figure below. ARPES matrix elements are calculated using hydrogenic orbitals.

The work function is assumed to be Cu=4.46 eV

More details about this web application can be found here.

How To Use:

  1. Go to set parameters.
  2. Select a lattice geometry.
  3. Select the orbital type for each atom in a cell of the chosen lattice
  4. Input orbital level and energy for each orbital
  5. Submit and wait for calcuations
  6. Click plot bands to plot ARPES bands through specified cuts.
  7. click plot DOS to plot DOS.
  8. Click plot contour to plot spectral function at fixed energy.
  9. Click ARPES datafile to download ARPES datafile.
